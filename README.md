# Support Bridge

> Internal web platform for managing and cataloging industrial support adapters and mounting components.

## 📌 About

Support Bridge is a web application developed for working with industrial support adapters and related connection elements, including:

- OTF
- OTP
- SF
- SF-2
- SP
- SP-2
- ZDF
- ZDF-2

The platform provides a convenient interface for viewing product information, managing the catalog, and organizing technical data.

> **Note:** Some files, business logic, and project resources have been removed from this repository due to commercial confidentiality.
